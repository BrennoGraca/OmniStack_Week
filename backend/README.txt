To help in first steps:
1 - To start the server type yarn dev in the terminal

The controllers are used to save the data on the database, in this case we're using mongoDB

Install in the project
1 - Express     ->  yarn add express
2 - Nodemon     ->  yarn add nodemon -D
3 - Mongoose    ->  yarn add mongoose
4 - Multer      ->  yarn add multer

Obs: -D is used to flag in packjson to use only in dev environment

Install in the machine to help in the project and future projects
1 - Insomnia.rest
2 - JsonViewer (chrome browser)
